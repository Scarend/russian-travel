# Проект: Путешествие по России

### Обзор
* Интро
* Figma
* Github Pages

**Интро**

Учебный проект - одностраничный сайт с адаптивным дизайном про внутренний туризм в России.

Стэк технологий:
 - HTML5
 - CSS3
 - БЭМ / файловая структура в соответствии с Nested БЭМ
 - Адаптивная верстка
 - Grid Layout и Flexbox-верстка
